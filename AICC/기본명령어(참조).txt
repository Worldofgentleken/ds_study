명령어
command [옵션] [인자]

pwd: 현재 디렉터리 확인

cd: 디렉터리 변경(이동)
	cd [옵션]
	path: 이동할 디렉터리 경로
	.: 현재 디렉터리
	..: 상위 디렉터리
	~: 로그인한 사용자의 홈 디렉터리

ls: 디렉터리 목록 확인
ls [옵션] [디렉터리/파일]
	(dir)
	-a: . 파일을 포함 모든 파일/디렉터리
	-l: 자세하게
	-d: 디렉터리 정보
	-F: 파일 형식 표현

cp: 파일/디렉터리 복사
cp [옵션] [source] [target]
	-r: 디렉터리 복사
	-b: 중복 파일 존재 시 백업 파일 생성
	-i: 중복 확인

mkdir: 디렉터리 생성
mkdir [경로]
	-p: 상위 경로 자동 생성

rmdir: 빈 디렉터리 삭제 (잘 사용하지 않음)

rm: 파일/디렉터리 삭제
rm [옵션] [대상 폴더/디렉터리]
	-r: 디렉터리 삭제
	-f: 강제 삭제
